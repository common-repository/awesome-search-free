== Description ==

Awesome Search is a WordPress plugin to make it easy to find text in your WordPress installation including in core files, WordPress plugins and your database.

= Solve WordPress Problems Easy =
It's the easiest way to find where text, code, images and other hard to find code is coming from.

= Fastest Way to Find Plugin & Theme Conflicts =
Find what you’re looking for and fast. Enter any string of text, and we’ll search your database, and analyze your files to tell you exactly where the text is including the fields or the line number.

= Start Free, Then Go Pro =
You can search files in the WordPress core, active plugins, your theme and more. Both paid and free options will return the full results of your file search and how many matches are in your database. For the full results, go pro!


== Installation ==

1. Download the .zip for the `<?php do_action('plugin_name_hook'); ?>` plugin.
2. Go to the Plugins section on your WordPress.
3. Press the "Add New" button at the top of the section.
4. Press the "Upload Plugin" button and then select the zipped file downloaded in the first step.
5. Finally, install and activate the plugin.

== Frequently Asked Questions ==

= Who is this for? =
Anyone who is looking to figure out where an SQL injection is coming from, where a piece of CSS is stored. You’ll find the right plugin to fix it fast.

= What does it do? = 
It searches your WordPress installation including root files, plugins and your database to help troubleshoot problems, fix conflicts and to figure out where code including scripts, CSS and more are coming from.

= Is there a free version? =
Yes, and it has all of the same features as the paid pro version, exceptonly limited database results (the # of matches, but not where they are) will be returned.

= Can I search only certain plugins? =
Yes, we’re constantly updating our database of plugins, and even if we don’t have them yet in our system – you can still search them.

= What if I only want to search files? = 
That’s an option.
